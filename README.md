# library-management-system-design
# Java /JDBC /MySQL /GUI

DESIGNED BY GU 

## Project Introduction
Library management system is building applications in java. This project focuses on how to create a library management system project in java. The system is divided into two roles: administrator and user. This system could be used in local bookstores and school library. The main feature of this system is that all the books available in the bookstore or school can be borrowed by users and also the borrowed books by users can be returned in the bookstore or school.
The main features of the administrator include create and reset databases, add users, view users, add books, issue books, view books, view issued books, and return books. The main features of ordinary user include viewing books and viewing his or her borrowed books.

## Technology
Java, Eclipse IDE, MySQL(open-source relational database management system), MySQL JDBC Connector, Databases, JDBC, GUI, Swing. multithread programming
